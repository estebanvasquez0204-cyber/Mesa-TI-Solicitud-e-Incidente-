# Mesa de Ayuda TI - Solicitudes e Incidentes

## Descripción del Proyecto
Este proyecto corresponde al desarrollo del sistema de **Mesa de Ayuda TI**, el cual permite registrar, atender y dar seguimiento a solicitudes e incidentes reportados por los usuarios de una organización.

El sistema facilita la comunicación entre los usuarios y el personal de soporte, permitiendo gestionar los casos de manera organizada, clara y eficiente.

## Objetivo del Proyecto
Desarrollar un sistema que permita:

- Registrar casos de soporte
- Consultar el estado de los casos
- Asignar y atender solicitudes
- Notificar al usuario sobre cambios
- Llevar control y seguimiento de la atención
